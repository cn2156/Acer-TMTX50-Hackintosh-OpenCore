# Acer TravelMate TX50 Hackintosh OpenCore

适用于 Acer TMTX50-G2-50D4

使用原来的 Intel 无线网卡，启动非图形界面

睡眠唤醒有问题：黑屏、不能正常唤醒

主要参考：https://dortania.github.io/OpenCore-Install-Guide/
